Image Processing Filters

This project is an image processing tool built using Python. It applies multiple filters to images using a modular architecture.

Features

* Grayscale conversion
* Thresholding
* Gaussian Blur
* Sharpening
* Sobel Edge Detection

Project Structure

```
src/
 └── modules/
     └── youssef_girguis/
         └── youssef_girguis_module.py

images/
 ├── original.jpg
 ├── grayscale.png
 ├── threshold.png
 ├── gaussian_blur.png
 ├── sharpen.png
 └── sobel_edge.png
```

How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the application:

```
python src/main_app.py
```
